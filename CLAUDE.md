# NVMe Driver Patcher v4.0.0

## Overview
PowerShell WinForms GUI tool for patching Windows 11 NVMe drivers. Async preflight checks via `[PowerShell]::Create()` + `BeginInvoke()`.

## Tech Stack
PowerShell, WinForms GUI (~3470 lines)

## Build / Run
